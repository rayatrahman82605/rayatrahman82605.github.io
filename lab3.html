Lab 3 - Responsive Design

An introduction to responsive web development using Bootstrap

About Responsive Design

Responsive web design lets a site adapt to different screen sizes — from desktops, tablets, to phones. With Bootstrap’s grid and helper classes, crafting a layout that works across devices becomes easier.

  • Use the Bootstrap grid system (rows & columns)
  • Use responsive images & typography
  • Test your layout at different viewport widths
